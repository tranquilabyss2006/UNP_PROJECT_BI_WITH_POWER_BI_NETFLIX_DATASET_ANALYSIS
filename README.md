# UNP_PROJECT_BI_WITH_POWER_BI_NETFLIX_DATASET_ANALYSIS

I did this project in collaboration with UNP, where the primary objective was to analyze netflix subscription data and derive meaningful business insights using Power BI.
This project focuses on understanding revenue analysis , customer subscription behavior, and trends in a subscription purchase environment. The dataset includes key attributes such as gender, revenue, subscription type, join date, and location-based data. Using Power BI, I transformed raw data into interactive and visually appealing dashboards to support data-driven decision-making.
The analysis highlights important KPIs such as total revenue, users, top-selling subscription plans, and their trends over time. I created dynamic visualizations like bar charts, line graphs, scatter plots, and slicers to enable easy filtering and exploration of data based on different parameters such as date, subscription type, age group, and country.
One of the key insights derived from this project includes identifying most subscribed user region, best-performing subscription plans, and customer-device preferences. Additionally, the dashboard helps in comparing revenue across different countries and understanding subscription patterns.
Overall, this project demonstrates how Power BI can be effectively used to convert raw sales data into actionable insights, helping businesses improve their subscription sales strategies, increasing revenue, and enhance customer userbase.
