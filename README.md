# UNP_PROJECT_BI_WITH_POWER_BI_NETFLIX_DATASET_ANALYSIS

I did this project in collaboration with UNP, where the primary objective was to analyze coffee sales data and derive meaningful business insights using Power BI.
This project focuses on understanding sales performance, customer purchasing behavior, and product trends in a coffee retail environment. The dataset includes key attributes such as product categories, sales revenue, quantity sold, time of purchase, and location-based data. Using Power BI, I transformed raw data into interactive and visually appealing dashboards to support data-driven decision-making.
The analysis highlights important KPIs such as total sales, profit, top-selling products, and sales trends over time. I created dynamic visualizations like bar charts, line graphs, and slicers to enable easy filtering and exploration of data based on different parameters such as date, product type, and region.
One of the key insights derived from this project includes identifying peak sales hours, best-performing coffee types, and customer preferences. Additionally, the dashboard helps in comparing sales across different locations and understanding seasonal demand patterns.
Overall, this project demonstrates how Power BI can be effectively used to convert raw sales data into actionable insights, helping businesses improve their sales strategies, optimize inventory, and enhance customer satisfaction.
